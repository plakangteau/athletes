
# Blueprint: Athlete Face Match

## 1. 개요

이 프로젝트는 사용자의 얼굴을 분석하여 어떤 유형의 운동선수와 가장 유사한지 알려주는 대화형 웹 애플리케이션입니다. 사용자는 웹캠을 통해 실시간으로 얼굴을 스트리밍하거나 사진 파일을 업로드할 수 있습니다. 애플리케이션은 제공된 얼굴 이미지를 분석하여 미리 정의된 운동선수 유형과 비교하고 가장 가능성이 높은 결과를 표시합니다.

## 2. 초기 디자인 및 기능 (v1)

### 구조
- `index.html`: 기본 HTML 구조.
- `style.css`: 시각적 스타일링.
- `main.js`: 핵심 애플리케이션 로직.

### UI 구성 요소
- **헤더**: 애플리케이션 제목과 간단한 설명.
- **입력 섹션**: 사용자가 웹캠 스트림을 보거나 이미지를 업로드할 수 있는 영역.
- **결과 섹션**: 분석 결과를 텍스트와 이미지로 표시하는 영역.
- **버튼**: "분석하기" 및 "다시 시도" 버튼.

### 스타일링
- **테마**: 현대적이고 활기찬 느낌을 주는 어두운 테마.
- **폰트**: 가독성이 좋은 Google Fonts의 'Poppins'.
- **레이아웃**: Flexbox를 사용한 반응형 레이아웃으로 모바일과 데스크톱 환경 모두 지원.
- **시각 효과**: 카드 UI, 부드러운 전환 효과 및 그림자를 사용하여 깊이감을 줍니다.

## 3. 현재 계획

1.  **`blueprint.md` 파일 생성**: 프로젝트의 목표와 계획을 문서화합니다.
2.  **HTML (`index.html`)**: 웹 애플리케이션의 기본 골격을 구성합니다.
3.  **CSS (`style.css`)**: 초기 스타일링을 적용하여 시각적 기반을 마련합니다.
4.  **JavaScript (`main.js`)**: 초기 스크립트 파일을 생성하고 HTML에 연결합니다.
5.  **기본 UI 구현**: 헤더, 입력 영역, 결과 표시 영역의 기본 구조를 완성합니다.
6.  **기능 구현 (단계적)**:
    *   **1단계 (현재)**: UI 레이아웃 완성 및 모의(mock) 결과 표시 기능 구현.
    *   **2단계**: 웹캠 API를 연동하여 실시간 비디오 스트림 표시.
    *   **3단계**: 이미지 업로드 기능 구현.
    *   **4단계**: 얼굴 인식 라이브러리 (예: `face-api.js`)를 통합하여 얼굴 감지.
    *   **5단계**: (모의) 분석 로직을 구현하여 "운동선수 유형"을 결과로 표시.
